# Financial Freedom OS

An intelligent financial freedom platform to track assets, calculate FIRE progress, and optimize tax intelligence.

## Structure

- `apps/web`: Next.js dashboard
- `apps/api`: Python backend with AI Oracle
- `apps/mobile`: Expo mobile application
- `packages/`: Shared logic and components

## Getting Started

1. Install dependencies: `npm install`
2. Run development servers: `npm run dev`
